-bootCD: collection d'outils permettant de créer 
très facilement des CD autobootables à partir de programmes Dreamcast 
(crée des iso ou des images DiscJuggler, Nero)
Pour l'utiliser placez dans Exemple votre programme 
(qui doit être un fichier *.bin scramble grâce à elf2bin) 
que vous aurez renommé 1st_read.bin, si vous avez d'autres fichiers 
(ou des répertoires) à placer, mettez tout dans Exemple 
(ne supprimez surtout pas Ip.bin!). Vous n'avez plus qu'à lancer MakeDj 
ou MakeNero pour avoir une image DiscJuggler ou Nero et le tour est joué!
(bien sûr ça marche pour tout: émulateur, divx etc...)
J'ai ajouté un chargeur d'application (Loader/), copiez ces fichiers
dans exemple (remplace le IP.BIN) pour choisir votre programme
au boot de la Dreamcast.
Pour créer un CD autobootable d'un jeu, créez une iso 
(avec Winiso par exemple http://www.winiso.com/),
puis patchez la avec bin2boot dans progs.

-elf2bin: pour convertir les fichiers *.elf en *.bin exécutables par la Dreamcast
l'opération scramble consiste à convertir le fichier *.bin  de sorte qu'il puisse être gravé
mais vous n'en avez pas besoin si vous souhaitez juste transférer votre programme
via le cable link de la Dreamcast

-DCTool: outils pour envoyer des programmes à la Dreamcast 
(utilisez la version normale si vous passez par le cable en COM 
ou alors la version IP si vous utilisez le modem)
Gravez sur CD les DCload.
Tapez dc-tool -x application.bin pour lancer un prog à la DC,
ou alors make run si vous compilez sous KOS.

-Serial Upload Slave: programme Dreamcast (comme DCTool) attendant la réception d'un programme,
image DiscJuggler, à graver donc avec ce logiciel (http://www.padus.com/), fourni
avec DCUp32 (apllication pour windows permettant d'envoyer des fichiers binaires 
(ou des fichiers SREC) à la Dreamcast)

-dcsrec: pour convertir un fichier *.bin en *.srec pour uploader via Serial Upload Slave
