IP Maker の使い方

［概要］
IP Makerは、Dreamcast用のアプリケーションディスクで必要な「イニシャル
プログラム」を生成するためのツールです。

［使い方］
IPMAKER.EXEをダプルクリックして実行してください。
IP Makerのメインウィンドウには全部で3つのタグがあり、起動時には、
一般設定タグの内容が表示されています(左上)。
もし、ダミーのイニシャルプログラムが欲しい場合には、この状態で画面右下
の「適用」ボタンを押してください。
また、IP Makerは以前の設定を保存しますので、一番最初の設定に戻したけれ
ば、「初期値に戻す」ボタンを押してください。

［各タグの説明］
・一般設定
  アプリケーションのタイトル
    アルファベットの大文字、スペース、数字、記号を使用し、96文字以内で
    アプリケーションのタイトルを入れます。
    アルファベットの小文字を入力した場合には、自動的に大文字に変換され
    ますのでご注意ください。
  SEGAタイトルの時にチェック
    SEGAブランドで販売するタイトルのイニシャルプログラムを作成する場合
    には、ここをチェックしてください。
  部品番号を使用する
    体験版などの目的で作成するディスクで商品番号の代わりに部品番号を
    使用する際には、ここにチェックを入れます。
    ここにチェックを入れると、下に、部品番号と書かれたフィールドが現れ
    ますので、SEGAの営業や、プロモーションから取得した部品番号をそこに
    入力します。
    部品番号を使用する際にも、商品番号のフィールドを埋める必要があります。
  日本以外の販売タイトル
    イニシャルプログラムの内部では、日本国内で販売されるタイトルと、
    日本以外の国で販売されるタイトルの区分けがなされています。もし、
    日本以外の国で販売されるタイトルのイニシャルプログラムを作成する場合
    には、ここにチェックを入れてください。
    ここにチェックを入れると、商品番号フィールドの初期値が、変化します。
  商品番号
    SEGAの営業から取得している商品番号を入力します。
  部品番号
    「部品番号を使用する」にチェックが入っている時のみ表示され、部品番号
    を入力できるようになります。部品番号を使用する際にも、商品番号は必要
    となりますので、必ず両方入力するようにして下さい。
  対応エリア
    発売されるタイトルが対応するエリアを最低でも一つチェックします。
    当然のことながら、例えばヨーロッパ対応ならばアプリケーションは
    PALモードに対応しているといったような、その地域の環境に合わせた対応
    がなされている必要があります。
  バージョン番号
    アプリケーションのバージョン番号を記述します。
    バージョンは、メジャーバージョン1桁、マイナーバージョン3桁です。
    マスターリリース時のバージョン1.000を境にバージョン番号を決めてくだ
    さい。
  リリース日時
    該当するアプリケーションディスクを納品する日時を記述してください。
  DreamcastOSを使用
    MicrosoftのOSベースのアプリケーションの場合、チェックしてください。
  VGAボックスに対応
    VGAモニタへの画面表示に対応したアプリケーションの場合、チェックして
    ください。
  ファーストリードファイル名
    GD-ROMから最初に読み込まれるファイルの名前を入れてください。
    ファーストリードファイル名は、アルファベットの大文字、数字、アンダー
    スコアが使用可能で、8文字以内の名前とピリオド、3文字以内の拡張子と
    いうフォーマットでファイル名の先頭は必ず数字で始まる必要があります。
    また、GD-ROMのルートディレクトリにある必要があり、アルファベット順に
    ファイル名を並べた時に、ファーストリードファイル名の前に0GDTEX.PVR
    以外のファイル名が来てはいけません。
  ディスク情報
    一つのアプリケーションが複数枚のディスクで構成されている場合、
    これから作るイニシャルプログラムが何枚目のディスク用のイニシャルプロ
    グラムであるのかを明示する必要があります。
    その場合、このフィールドの初期値は使用せず、適切な数値を入力する
    必要があります。

・ペリフェラル設定
  このタグでは、アプリケーションの対応するボタン、ペリフェラルに関しての
  情報を設定します。
  アプリケーションを動作させるのに最低限必要なボタン
    アプリケーションを動作させるために最低限必要なボタンをすべてチェック
    します。
    仮に、Dreamcast Controller(標準パッド)と、アプリケーションに特化した
    別のペリフェラル(例えばアーケードスティック)では、操作体系が異なると
    いった場合には、Dreamcast Controllerの設定のほうを記述してください。
  対応しているペリフェラル
    リストに上がっているペリフェラルに対応しているようであればチェックを
    入れてください。
  対応している拡張ペリフェラル
    ペリフェラルの拡張スロットに挿すような種類のペリフェラルで、対応して
    いるものがあるようならばチェックを入れてください。

・プレイ履歴情報
  イニシャルプログラムにプレイ履歴の書き込み機能を付加する時に使用します。
  プレイ履歴を保存する
    ここにチェックが入ると、イニシャルプログラムが実行時にプレイ履歴情報
    を保存します。チェックを入れると、下に、2つの項目が表示されます。
  プレイ履歴で使用するタイトル名
    プレイ履歴のシステムでは、イニシャルプログラムのアプリケーションタイ
    トルとは別に、アプリケーションの名称を持つことが出来ます。
    その際に日本語を使用することも可能です。
    タイトルは44バイト(アルファベットのみなら44文字、全角の日本語のみなら
    22文字)まで認められます。
  アプリケーションのジャンル
    アプリケーションの該当するジャンルをすべてチェックします。

【コンフィギュレーション画面】
画面の左下に「Config」と書かれたボタンがあります。
このボタンを押すとツールの設定を行うことが出来ます。
・言語設定
  ツールの表示言語を変更します。
・データベースディレクトリのパス
  IP Makerは、書き込まれた内容やチェックされた内容を元に、複数のパーツを
  結合させて一つのイニシャルプログラムを作成します。
  これらのパーツのあるディレクトリをデータベースディレクトリと呼び、
  通常は、パッケージの中のBINディレクトリがそれに当たります。
  もし、IPMAKER.EXE本体を何処か別の場所に移動した時には、データベース
  ディレクトリの場所を正確にツールに反映させる必要があります。
  そのための設定をするのがこのフィールドになります。
・出力ファイルの設定
  IP Makerは、出力結果として、単密用と高密用の二つのイニシャルプログラム
  出力が可能です。
  この名前はデフォルトでそれぞれ、IP0000.BINとIP.BINですが名前が決まって
  いるわけではなく、自分の好きなように変えることが出来ます。
