Direct3D sample program -- MMatrix
-----------------------------------------

* Purpose:

This sample demonstrates how to utilize the custom Multi-matrix functionality
that has been added to DrawIndexedPrimitive. The sample animates thirty-six
texture-mapped bananas. Each banana utilizes 22 transformation matrices and
is rendered through a single call to DrawIndexedPrimitive.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
