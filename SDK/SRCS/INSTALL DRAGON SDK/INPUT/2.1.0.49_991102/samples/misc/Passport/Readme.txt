Miscellaneous sample program -- Passport
-----------------------------------------

* Purpose:
This sample demonstrates how to retrieve Dreamcast ID and passport information.
After reading the information, it outputs it to the debugger.

Additionally, it demonstrates how to load the passport information into the registry -
note that this is not necessary; it is included to demonstrate how to reload the
registry for applications that modify it at run-time.

* Required Files\Parameters:
This sample requires a valid Sega Passport, which can be created by Sega DreamPassport.
Please refer to Sega DreamPassport manual for instructions.

* This sample exits automatically after performing the specified functionality.
