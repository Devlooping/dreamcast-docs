MapleDevices sample program -- SaveGame
-----------------------------------------

* Purpose:
This sample demonstrates how to build a savegame system, detect VMU & controller
removal, VMU usage ( fast enumaration, using the cache, opening files using 
file name... ) and how to block dreamcast from rebooting if the GDRom door is 
opened at bad time ( while writing to VMU ).

* Required Files\Parameters:
None

* To make a file selection use DCTool virtual keyboard or a Dreamcast keyboard

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the virtual keyboard
