GDROM sample program -- Playtrack
-----------------------------------------

* Purpose:
This sample loads and plays a song from a CDDA track on the GD.

* Required Files\Parameters:
Usage: 
    s playtrack [options] <starting track number> [<ending track number> [<repeats>]]

    Options:
        -s <left vol> <right vol>   - Set volume to specified values
        -g                          - Display current GD-DA volume
        -w                          - Wait until playback is finished before exiting
        -?                          - Print usage

* After playing the track(s), the app automatically exits
