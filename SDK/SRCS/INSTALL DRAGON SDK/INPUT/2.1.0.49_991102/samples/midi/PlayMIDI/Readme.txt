MIDI sample program -- PlayMIDI
-----------------------------------------

* Purpose:
This sample shows how to play MIDI songs on WinCE for Dreamcast.  It
loads a MIDI file and associated tonebanks from it's resource file and
plays them back.

* Required Files\Parameters:
Usage:    s PlayMIDI <MIDI filename>

Example:  s PlayMIDI canyon.mid
Example:  s PlayMIDI \PC\Samples\canyon.mid

* This sample exits automatically after performing the specified 
functionality.
