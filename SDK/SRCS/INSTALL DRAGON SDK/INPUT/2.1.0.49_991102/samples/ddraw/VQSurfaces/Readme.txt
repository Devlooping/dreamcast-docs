DirectDraw sample program -- VQSurfaces
-----------------------------------------

* Purpose:
This sample shows how to use VQ-compressed surfaces in WinCE for
Dreamcast.  It loads three VQ images (RGBA 5650, RGBA 5551, and
RGBA 4444) from the resource file, creates compressed surfaces
for them, and displays the surfaces on the screen.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
