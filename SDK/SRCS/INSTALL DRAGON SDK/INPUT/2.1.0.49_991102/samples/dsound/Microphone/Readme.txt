DirectSound sample program -- Microphone
-----------------------------------------

* Purpose:
This sample demonstrates how to use the Microphone maple device on Dreamcast.
It records for a second, and then plays back what it recorded, and then repeats.
It does this for 100 seconds, and then exits.

The sample only works with a microphone plugged into the controller in port 0.

* Required Files\Parameters:
None

* This sample plays for 100 seconds and then automatically exits.
