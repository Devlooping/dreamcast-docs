DirectPlay sample program -- PeerToPeer
-----------------------------------------

* Purpose:
This sample demonstrates how to set up and use a peer-to-peer network
connection.  One of the peers acts as the host of the session, the other
peers connect to the host and all exchange messages every second.
To view the messages, make sure to run the sample under a debugger
and watch the Ouput window.
This sample only works over the Winsock service provider.

* Required Files\Parameters:
To start the host, type 's PeerToPeer'
To join a session, type 's PeerToPeer <hostname or ip address>'

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
