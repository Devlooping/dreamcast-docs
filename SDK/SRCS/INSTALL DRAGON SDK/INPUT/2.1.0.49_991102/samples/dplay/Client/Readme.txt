DirectPlay sample program -- Client
-----------------------------------------

* Purpose:
This is the Client side of the ServerClient DirectPlay sample.
This sample, used in conjonction with the Server sample, 
demonstrates how to set up and use a client-server network
connection.  The Server sample acts as the host of the session, while the 
clients connect to the host and exchange messages every second.
To view the messages, make sure to run the client under a debugger
and watch the Ouput window.
This sample only works over the Winsock service provider.

* Required Files\Parameters:
To start the host, type 's Server'
To join a session, type 's Client <hostname or ip address of machine running the Server>'

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
