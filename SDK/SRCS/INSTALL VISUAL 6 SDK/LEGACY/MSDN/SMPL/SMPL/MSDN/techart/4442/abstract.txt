GRPHDEMO: Uses Microsoft Graph as an OLE Automation 
Server
The GRPHDEMO sample application demonstrates the use of Microsoft(R) Graph as an OLE Automation 
server. This sample uses Microsoft Visual Basic(R) version 4.0 as the OLE Automation controller. 
Developers can use this sample to gain a better understanding of how to implement OLE Automation with 
Microsoft Graph.
GRPHDEMO is a companion sample for the "Using Microsoft OLE Automation Servers to Develop 
Solutions" technical article in the Microsoft Development Library.
Note that you must have Microsoft Graph and its associated type libraries fully installed on your system to 
run this sample.
The GRPHDEMO sample was built with the Enterprise Edition of Microsoft Visual Basic version 4.0 and 
tested on Microsoft Windows(R) 95.
KEYWORDS: CD14; OLE; VBA
