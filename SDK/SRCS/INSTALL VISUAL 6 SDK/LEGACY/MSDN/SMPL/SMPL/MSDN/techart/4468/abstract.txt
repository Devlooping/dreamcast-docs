NTDDWZD: Implements a Windows NT Kernel-Mode Device 
Driver Wizard
The NTDDWZD sample implements a custom wizard for Microsoft(R) Windows NT(TM) kernel-
mode device drivers. This sample accompanies the technical articles "Wizards Simplify 
Windows NT Kernel-Mode Driver Design" and "Using the Windows NT Custom Driver Wizard" 
in the Microsoft Development Library.
NTDDWZD was built with Microsoft Visual C++(TM) version 4.0 and tested on Windows NT 
version 3.51.
