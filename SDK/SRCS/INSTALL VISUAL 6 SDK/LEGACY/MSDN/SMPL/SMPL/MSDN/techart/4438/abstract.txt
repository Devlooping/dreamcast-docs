BINDDEMO: Uses the Microsoft Office 95 Binder as an OLE 
Automation Server
The BINDDEMO sample application demonstrates the use of the Microsoft(R) Office 95 Binder as an OLE 
Automation server. This sample uses Microsoft Visual Basic(R) version 4.0 as the controller. Developers 
can use this sample to gain a better understanding of how to implement OLE Automation with the Microsoft 
Office 95 Binder.
BINDDEMO is a companion sample for the "Using Microsoft OLE Automation Servers to Develop 
Solutions" technical article in the Microsoft Development Library.
Note that you must have Office 95 and its associated type libraries fully installed on your system to run this 
sample.
The BINDDEMO sample was built with the Enterprise Edition of Microsoft Visual Basic version 4.0 and 
tested on Microsoft Windows(R) 95.
KEYWORDS: CD14; OLE; VBA
