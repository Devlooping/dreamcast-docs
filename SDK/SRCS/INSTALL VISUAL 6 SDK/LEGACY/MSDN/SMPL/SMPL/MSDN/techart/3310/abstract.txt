METAVIEW: An Enhanced Metafile Viewer Using MFC
METAVIEW is an enhanced metafile viewer written using the
Microsoft(R) Foundation Class Library (MFC). It is a
companion sample to the "MFC: I'll Never Use It!"msdn_mfcexp
technical article in the Microsoft Development Library. The
sample demonstrates how to open and view an enhanced
metafile in two views: the rendered image or the metafile
header.
METAVIEW was built with Microsoft Visual C++(TM) version 2.1
and MFC version 3.1, and tested under Microsoft Windows
NT(TM) version 3.5.
