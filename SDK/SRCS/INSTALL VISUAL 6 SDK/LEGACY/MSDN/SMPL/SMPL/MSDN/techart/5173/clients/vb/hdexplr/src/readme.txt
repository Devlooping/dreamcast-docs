HelpDesk Sample Explorer readme.txt as of Jan 5/98

REQUIRED TO RUN
	As the Helpdesk is an MTS sample the DBExec/HDServer components should be instantiated on the remote server.

STILL UNDER CONSTRUCTION
The following features are known to be incomplete and will be finished in the final version.

1)	Domain maintenance will display the information but not allow for updates or deletes.

2)	Technician maintenance will display the information but not allow for updates or deletes.

3)	The toolbar state has not been finalized.
