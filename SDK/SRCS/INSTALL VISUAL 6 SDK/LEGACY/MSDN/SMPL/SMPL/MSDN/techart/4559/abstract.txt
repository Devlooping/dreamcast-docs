THRDPERF: Tests Multithreading Performance
The THRDPERF project consists of two parts: A Microsoft(R) Win32(R) application, Threadlibtest.exe, 
and a class library, Thrdlib.dll. Threadlibtest is used to sample multithreading data. Please refer to the 
article "Win32 Multithreading Performance" for details on how to build the code samples. 
This sample was built with Microsoft Visual C++(R) version 4.0 and tested on Microsoft Windows 
NT(TM) version 3.51 and Windows(R) 95.
