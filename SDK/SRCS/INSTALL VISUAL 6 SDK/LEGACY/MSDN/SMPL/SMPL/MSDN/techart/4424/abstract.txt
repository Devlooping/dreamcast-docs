PAGETEST: Tests DLL Loading Performance
The PAGETEST project is a sample suite that tests dynamic-link library (DLL) loading performance on 
multithread Microsoft(R) Win32(R) systems. The PAGETEST project consists of two parts: the Win32 
sample application PTAPP, and a set of DLLs that can be customized for size, reload item count, and other 
parameters. PAGETEST accompanies the "Rebasing Win32 DLLs: The Whole Story" technical article in 
the Microsoft Development Library, which provides details on how to build the DLL samples.
The PAGETEST sample was built with Microsoft Visual C++(TM) version 2.2, and tested under Microsoft 
Windows NT(TM) version 3.51 and Windows 95.
