ANIMTEST: Tests Features of the ANIMATE Library
The ANIMTEST sample is a test application that exercises various elements of the ANIMATE sample 
library. ANIMTEST is a companion sample for the "Recording and Playing Waveform Audio" technical 
article in the Microsoft(R) Development Library.
Each application was built using the "MFC in a shared DLL" option, so you need the Microsoft Foundation 
Class library (MFC) dynamic-link libraries (DLLs) to run the samples. To run the release versions, you 
need MFC40.DLL and SVCRT40.DLL. To run a debug version, you need MFC40D.DLL and 
MSVCR40D.DLL.
Note that all platforms were running their display drivers in 256-color mode (8 bits per pixel) unless 
otherwise noted in the technical article that accompanies this sample.
The ANIMTEST sample was built with Microsoft Visual C++(TM) version 4.0 and tested on Microsoft 
Windows NT version 3.51, Windows 95, and Windows(R) for Workgroups version 3.11 with Microsoft 
Win32s(R) version 1.30 installed. Samples that use the ANIMATE sample library also require the WinG 
library version 1.0.
