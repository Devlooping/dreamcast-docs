OFFDEMO: Uses the Microsoft Office 95 Object as an OLE 
Automation Server
The OFFDEMO sample application demonstrates the use of the Microsoft(R) Office 95 object as an OLE 
Automation server. This sample uses Microsoft Visual Basic(R) version 4.0 as an OLE Automation 
controller. Developers can use this sample to gain a better understanding of how to implement OLE 
Automation with the Microsoft Office 95 object.
OFFDEMO is a companion sample for the "Using Microsoft OLE Automation Servers to Develop 
Solutions" technical article in the Microsoft Development Library.
Note that you must have Microsoft Office 95, Microsoft Project 95, and their associated type libraries fully 
installed on your system to run this sample.
The OFFDEMO sample was built with the Enterprise Edition of Microsoft Visual Basic version 4.0 and 
tested on Microsoft Windows(R) 95.
KEYWORDS: CD14; OLE; VBA
