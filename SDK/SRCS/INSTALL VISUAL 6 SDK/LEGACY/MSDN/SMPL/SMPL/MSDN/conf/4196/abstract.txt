 
 
 MM202: Demonstrate Console and Simple Windows Programming
 Under Windows NT
 
 These sample applications augment the "Microsoft Windows
 NT as an Engineering Platform" technical presentation
 given by Nigel Thompson at the Tech*Ed Conference held
 March 27-31, 1995, in New Orleans, Louisiana.
 
 The MM202 samples illustrate console applications, and
 simple Microsoft(R) Windows(R)-based applications
 developed under Windows NT(TM). The console samples
 include a moving train, a dropping ball, and a "Hello
 World" application. The simple Windows-based applications
 include an FM radio transmission application and a basic
 text editor.
