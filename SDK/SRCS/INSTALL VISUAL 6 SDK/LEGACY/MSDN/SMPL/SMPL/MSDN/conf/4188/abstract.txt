 
 
 FX203: Demonstrate Visual FoxPro Front End on SQL Server
 for Windows NT
 
 The FX203 sample application augments the "Creating Client-
 Server Solutions with Microsoft Visual FoxPro" technical
 presentation given by Joe Homnick at the Tech*Ed
 Conference held March 27-31, 1995, in New Orleans,
 Louisiana.
 
 All examples and sample code in the presentation use
 Microsoft(R) SQL Server for Windows NT(TM) version 4.21 as
 a back end and reference the demonstration "pubs" database
 that comes with SQL Server for Windows NT. The
 presentation and samples demonstrate a Visual FoxPro front
 end to a SQL Server.
