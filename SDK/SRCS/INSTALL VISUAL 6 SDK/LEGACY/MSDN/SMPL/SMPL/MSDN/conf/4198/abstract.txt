 
 
 NT213: Demonstrate Windows Sockets Interface
 
 The NT213 sample applications augment the "Developing
 Transport-Independent Applications Using the Windows
 Sockets Interface" technical presentation given by David
 Treadwell at the Tech*Ed Conference held March 27-31,
 1995, in New Orleans, Louisiana.
 
 These samples illustrate a usable open networking standard
 developed by the networking community. The Microsoft(R)
 Windows(R) Sockets interface defines an abstraction layer
 for the transmission internet protocol/internet protocol
 (TCP/IP), or Internet protocol family.
