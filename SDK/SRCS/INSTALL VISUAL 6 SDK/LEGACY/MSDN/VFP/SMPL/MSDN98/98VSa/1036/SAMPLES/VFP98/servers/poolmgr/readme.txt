Voici un exemple de Gestionnaire de groupes utilisant des serveurs OLE personnalisés. Vous (le client) pouvez envoyer des tâches à ce Gestionnaire de groupes (serveur OLE personnalisé VFP) qui peut être distant et lui faire gérer le traitement de ces tâches. Ce faisant, vous pouvez continuer à travailler puisque le Gestionnaire de groupes traite la totalité des tâches. De plus, il n'est pas nécessaire de vérifier l'accomplissement du travail à intervalles réguliers car le Gestionnaire de groupes vous rappelle lorsqu'il a terminé. Les tâches présentées sont des simulations.

Remarque: Excel est nécessaire pour une option de travail comportant un graphique.

Procédure à suivre:

1. Assurez-vous que vous avez bien généré le fichier POOL.EXE à partir du Projet de façon à ce qu'il soit enregistré comme serveur OLE personnalisé.

2. Exécutez le formulaire POOL.
