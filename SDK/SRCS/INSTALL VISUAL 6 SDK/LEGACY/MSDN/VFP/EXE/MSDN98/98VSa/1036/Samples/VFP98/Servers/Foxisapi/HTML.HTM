Démonstration du serveur Fox FOXISAPI!

Essayez la PAGE DÉMO DU SERVEUR OLE PERSONNALISÉ DE VFP. Les modifications apportées par l'utilisateur peuvent être héritées à la fois par les versions HTML et Fox du formulaire!
