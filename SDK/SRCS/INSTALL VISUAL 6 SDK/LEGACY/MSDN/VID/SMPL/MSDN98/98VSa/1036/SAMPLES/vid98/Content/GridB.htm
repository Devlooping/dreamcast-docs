L'envers du décor

Cet exemple utilise un seul fichier : Grid.asp. Ce fichier utilise deux contrôles DTC majeurs.

Le contrôle Recordset permet de se connecter à la table Products de la base de données Gallery.

Le contrôle DTC Grid est lié au jeu d'enregistrements pour afficher tous les enregistrements de la table.

Cet écran contient quatre boutons qui appellent certaines des méthodes exposées par le contrôle Grid.

  • Le premier bouton appelle la méthode de masquage (hide) afin de rendre la grille invisible.
  • Le deuxième bouton appelle la méthode d'affichage (show) afin de rendre la grille de nouveau visible.
  • Le troisième bouton appelle la méthode isVisible et génère un message d'alerte indiquant à l'utilisateur si la grille est visible ou pas.
  • Le dernier bouton appelle la méthode bindAllColumns qui affiche chaque colonne de la grille si toutes les colonnes ne sont pas affichées.
