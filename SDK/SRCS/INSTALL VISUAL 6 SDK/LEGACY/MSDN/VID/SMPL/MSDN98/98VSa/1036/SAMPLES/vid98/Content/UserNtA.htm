Cet exemple emploie l'authentification des utilisateurs pour l'échange de messages avec d'autres utilisateurs ayant accès à cette page.

Remarque Pour que cet exemple fonctionne correctement, vous devez désactiver l'accès anonyme au dossier UserNote. Pour ce faire, utilisez Microsoft Management Console. Cliquez avec le bouton droit sur le dossier UserNote dans le projet Gallery et sélectionnez Propriétés. Le contrôle par authentification peut être adapté à l'aide de l'onglet Sécurité de répertoire de la boîte de dialogue Propriétés.
