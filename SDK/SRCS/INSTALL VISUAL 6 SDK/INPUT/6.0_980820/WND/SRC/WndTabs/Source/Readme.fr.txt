Installation
------------
To use this language pack follow these steps:
1. Copy the .dll file to the WndTabs installation directory (usually 
   "C:\Program Files\WndTabs").
2. Restart Visual C++.
3. In the WndTabs options dialog, pick "Language Selection" in the tree on
   the left hand side.
4. The new language should appear in the combo box.

Please note that this language pack will work ONLY with the version of
WndTabs specified at the top of this file.

-----------------------------------------------------------------------------

Installation
------------

Pour utilisez ce package suivre les étapes suivantes :
1. Copier le fichier .dll dans le répertoire d'installation de WndTabs (généralement 
   "C:\Program Files\WndTabs").
2. Redémarrer Visual C++.
3. Dans la boite de dialogue "Options" de WndTabs, choisissez "Choix de la langue" 
   dans l'arbre à gauche.
4. La nouvelle langue devrait apparaître dans la liste déroulante.
Attention cette traduction fonctionnera uniquement avec la version de WndTabs
indiquée au début du document.
