Customize|Keyboard (Options Dialog Box)

Options

Assign Keyboard Shortcuts

Set Tab Activation Shortcuts

Opens a dialogs from which you can assign keyboard activation keys for tabs 1..9.

Set Shortcuts for Common Commands

Opens a dialog through which you can assign keyboard shortcuts to common WndTabs commands.

Please note that unlike previous versions, WndTabs now manages its own keyboard shortcuts.  If you select a key combination for a WndTabs shortcut, it will take over any DevStudio shortcut with the same combination (resetting the WndTabs shortcut will restore the functionality of the DevStudio shortcut).

Set Shortcuts for Any Command

Clicking on this button opens the advanced keyboard customization dialog.  Through this dialog, you can assign a shortcut (or multiple shortcuts) to any WndTabs command. 

Internet Keyboards

These options allow you to control the special keys found on Internet keyboards.  You can right click on each option to receive further help.
