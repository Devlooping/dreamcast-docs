The WndTabs Extension Module (WndTabsExt)

Freeware and Shareware In One Package

WndTabs started out as a cross between what I though was a much needed addition to DevStudio, and a sample for MFC/Win32/Add-In programming. Both WndTabs and it's source were released to the public - for free.

WndTabs is a large, fast evolving project. I spend an enormous amount of time answering your e-mails, and fixing/updating the code to your requests. Unfortunately, keeping WndTabs a high-standard add-in is costing me more time and money than I can afford. In order to offset the financial and personal costs, I introduced the WndTabs Extension Module (WndTabsExt) in version 2.50.  WndTabsExt is a shareware module which adds many advanced functions to WndTabs. (Registration information is here).

Advanced Functionality - Overview

Registered users of WndTabsExt have access to many advanced features not available in the freeware version.  

  • The Advanced Tab Control with Tab Grouping and coloring
  • Full UI (menus + keyboard) customizations
  • Access to full range of WndTabs commands
  • Enhanced functionality for addons: tab customizations

Throughout the WndTabs online documentation, theicon is used to identify WndTabsExt specific features.  These features are not available to non registered users.

The Advanced Tab Control

The advanced control was designed with two purposes in mind:

  1. To save screen real-estate.
  2. To help make tabs and file more manageable.

These goals are achieved through two primary features:

  1. Grouping: The advanced tab control is capable of grouping multiple files on a single tab using a proprietary mini-toolbar.  This allows more tabs to fit on the screen, and also allows you to control the entire group of files with a single command.



  2. Coloring: The advance tab control allows you to choose different colors for selected and non selected tabs.  
  3. Quick Close ("X") Button: Lets you close the tab with a single click or multiple tabs by right-clicking.
  4. Custom Transformations: WndTabs Addons can modify the tabs in many ways.  For instance, in the above image, icons read only files were given a red background by the WndTabs SDK sample addon.  You can write your own addons using the WndTabs SDK.

For more information about these features, read here.

Full UI Customizations

WndTabsExt users have full control over the WndTabs UI.  This includes customizations of toolbars and menus as well as full customizations of keyboard shortcuts.  Using these versatile features, you can craft a version of WndTabs that exactly suited to your needs.

Other Goodies

WndTabsExt offers some other smaller features which are sprinkled throughout the program.  These features are clearly marked in the WndTabs option dialogs.
