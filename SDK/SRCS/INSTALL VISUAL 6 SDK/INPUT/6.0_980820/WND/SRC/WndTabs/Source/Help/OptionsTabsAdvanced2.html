Tabs|Advanced #2 (Options Dialog Box)

Overview

This dialog allows you to set options for the advanced tab control.  The advanced tab control is available only to registered users of WndTabsExt.

Options

Other Styles/Options

Add close button to each tab

With this option enabled, an "X" icon appears on the right side of the tab.  Clicking on this icon closes the tab and its associated windows.

   


Tip: If you right-click on the "X" a special context menu will appear. 

Tabs without a group look like grouped tabs

This options forces all tabs to use the mini-toolbar.  Without this option, tabs that have no group appear as normal tabs.

   

Tab Colors

With these selectors you can choose colors for selected and regular tabs.
