Tabs|Trimming (Options Dialog Box)

Overview

This dialog allows you to set various trimming options for the tabs.  Using trimming, you can shorten the length of the tabs, thus enabling more tabs to fit on the screen.

Options

Overall Trimming

Limit total number of characters on tabs

Use this option to limit the overall number of characters a tab can display.

Trim resource files

When checked, tabs of resource windows are also trimmed. Alternately, you can use the resource tab options in the tab content dialog to control the amount of information that goes on the resource tab in the first place.

Ignore file extension in character count

When checked, the length of the file's extension is not included in the count of characters being trimmed. That is - the extension is always shown, and only the file name is trimmed.
Has no effect on resource tabs.

Append extra character when trimming

With this option enabled, a character of your choice is appended to trimmed tabs. Disable this option if you don't want a character to indicate that a file name was trimmed. The appended character is included in the maximum tab length count (as specified by the "Limit total number of characters" option), effectively reducing the number of displayable file name characters by one.

Prefix Trimming

Trim the following prefixes...

In this edit box you may specify a comma delimited list of prefixes that you wish to strip from file names.  For example, if you specify "wwhiz,foo", the file "WWhizInterface" will be stripped of its leading "WWhiz", and the file "foobar" will be stripped of its leading "foo".

This option is especially useful if most of the files in your project begin with the same prefix.

Replace with the following character...

When enabled, the character of your choice will replace stripped prefixes.  For example, with the following settings:

Prefixes To Strip: WWhiz
Prefix Character: »

The file "WWhizInterface" will be displayed as "»Interface".

Suffix Trimming

Save tab numbers when closing the project file

The suffix trimming options are identical to their prefix trimming counterparts.  Please refer to the above explanations for further detail.
