Customize|Mouse (Options Dialog Box)

Options

Middle mouse button action

By clicking on the 'Assign' button, you can select any WndTabs command to be mapped to the middle mouse button.  For instance, if you select the "Close Window" command, clicking on a tab with the middle button will close that tab.
This feature does not affect the behavior of the middle mouse button outside of the tab area.

Shell context menu shortcut

This shortcut combination allows you to access a file's shell context menu directly.
