Customize Keyboard (Customize Dialog Box)

Overview

The Commands tab of the Customize Dialog allows you to assign custom keyboard shortcuts to all WndTabs commands.  To assign a keyboard shortcut, select a command from the commands list, and enter shortcut in the New Shortcut area.  You may assign multiple shortcuts to a single command.

WndTabs keyboard shortcuts take precedence over DevStudio shortcuts and keystrokes.  If you assign a WndTabs shortcut that has the DevStudio shortcut/keystroke, only the WndTabs command will be issued.  Be careful not to assign single-key combinations (such as "A") to WndTabs commands.  This will render them inoperable.

Alternate Shortcut Management Facilities

There are two other ways within WndTabs to manage keyboard shortcuts:

  1. The "Set Keyboard Shortcuts" dialog, available through the Options|General dialog allows setting of common WndTabs commands.  This dialog (available to both registered and unregistered users) can set up multiple shortcuts with a single command.
  2. Through Options Tab of the Customize dialog you can reset all your assigned keyboard shortcuts.
