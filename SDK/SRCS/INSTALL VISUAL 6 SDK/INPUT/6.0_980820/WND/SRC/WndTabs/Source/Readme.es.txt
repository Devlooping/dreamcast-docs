 Installation
 ------------
 To use this language pack follow these steps:
 1. Copy the .dll file to the WndTabs installation
 directory (usually 
    "C:\Program Files\WndTabs").
 2. Restart Visual C++.
 3. In the WndTabs options dialog, pick "Language
 Selection" in the tree on
    the right hand side.
 4. The new language should appear in the combo box.
 
 Please note that this language pack will work ONLY
 with the version of
 WndTabs specified at the top of this file.
 

-----------------------------------------------------------------------------
 
 Instalación
 -----------
 Para usar este paquete de lenguaje siga los siguientes pasos:
 1. Copie el archivo .dll al directorio donde WndTabs fue 
 instalado 
	(usualmente "C:\Program Files\WndTabs").
 2. Reinicialice Visual C++.
 3. En el diálogo de opciones de WndTabs, elija "Selección de Lenguaje"
 del árbol ubicado en el lado izquierdo.
 4. El lenguaje debería aparecer en el combo de opciones.

 Por favor note que este paquete de lenguaje SOLO funcionará
 con la versión de WndTabs especificada al comienzo de este archivo.
