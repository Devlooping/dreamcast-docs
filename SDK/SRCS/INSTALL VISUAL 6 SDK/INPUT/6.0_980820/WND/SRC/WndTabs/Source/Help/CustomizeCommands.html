Customize Commands (Customize Dialog Box)


Overview

The Commands tab of the Customize Dialog allows you to customize the WndTabs menus and toolbars in the same way you would with DevStudio and its own customize dialog.  For a list of all available WndTabs commands and their descriptions, click here.

The WndTabs toolbar - the area to the left of the tab - is available for adding more commands and menus.  You can even choose to remove the standard options menu if you want.

Customizations are performed by dragging commands from the commands list onto the toolbar area or the tab context menu.  You can also alt+drag existing items.


New Menus

You can extend the toolbar and tab context menu with new commands, as well as new menus.  The following section explains how to create such a menu:

STEP 1

Create the new menu by dragging the "New Menu" command to your desired destination.

Please note that although the example shows a new menu being created on the toolbar, the same procedure applies to the tab context menu.

 

 

 

 
STEP 2

Right click on the "New Menu" button created for the new menu.  From the various options, select "Image".

 
STEP 3

The Button Appearance dialog appears.  Select an image for the menu button.

 
STEP 4

You can now add items to the new menu, by dragging commands over it.  Once a command is dragged over the menu, the menu will open and you will be able to insert the command into the menu.

You can also click on the menu at any time in order to open and edit it.
