Installation et configuration du niveau A de Island Hopper News

Cette rubrique décrit la configuration requise et les instructions d'installation pour le niveau A de l'exemple Island Hopper News. Vous pouvez installer le niveau A sur un ordinateur.

Configuration requise

Le niveau A de l'exemple Island Hopper News n'exige aucune configuration matérielle spéciale en dehors de la configuration minimale requise pour Visual Studio 6.0.

Configuration logicielle requise

Avant d'installer le niveau A, vous devez avoir installé les logiciels suivants :

  • Visual Studio 6.0, Édition Professionnelle
  • Windows NT Option Pack, version 4.0
  • Internet Explorer 4.0 ou version ultérieure

Autres informations

L'Assistant d'installation vous invite à préciser l'emplacement où installer le contenu web. Vous pouvez accepter l'emplacement proposé ou spécifier votre propre serveur.

Vous ne pouvez pas installer les niveaux A et C de l'exemple sur le même ordinateur. Avant d'installer un niveau de l'exemple, vérifiez si l'autre niveau est complètement désinstallé.

Instructions d'installation

Pour installer Island Hopper News Niveau A :
  1. Insérez votre CD-ROM MSDN dans le lecteur.
  2. Naviguez jusqu'à samples\vs98\SampleA.
  3. Exécutez SetupA.exe. Ce programme est un fichier ZIP auto-extractible qui décompresse et copie les fichiers d'installation nécessaires sur votre disque dur, puis lance le programme d'installation de Island Hopper A.

Le niveau A de Island Hopper News est désormais installé sur votre système. Le code source se trouve sous Program Files\IslandHopper\ScenarioA. Vous pouvez exécuter au choix le client Windows ou Web pour l'application.

Pour démarrer le client Windows :

  1. Cliquez sur Démarrer.
  2. Pointez sur Programmes.
  3. Pointez sur Island Hopper.
  4. Cliquez sur Island Hopper A.

Pour démarrer le client Web :
  1. Démarrez Internet Explorer.
  2. Dans la zone Adresse, tapez le nom du serveur Web où vous avez installé l'exemple Island Hopper et le dossier pour le client Web (islandhoppera). Par exemple, http://monserveur/islandhoppera.

Pour plus d'informations, consultez les rubriques en ligne suivantes dans la bibliothèque MSDN Visual Studio 6.0 :

  • Exemple Island Hopper News
  • Rechercher et exécuter le niveau A
  • Spécifications du niveau A

Instructions de désinstallation

Pour désinstaller le niveau A :
  1. Insérez votre CD-ROM MSDN dans le lecteur.
  2. Naviguez jusqu'à samples\vs98\SampleA.
  3. Exécutez RemPackA.exe. Ce programme supprime le package IsleHop de Microsoft Transaction Server.
  4. Cliquez sur Démarrer.
  5. Pointez sur Paramètres.
  6. Cliquez sur Panneau de configuration.
  7. Double-cliquez sur Ajout/Suppression de programmes.
  8. Cliquez sur l'onglet Installation/Désinstallation.
  9. Sélectionnez Island Hopper Scenario A.
  10. Cliquez sur Ajouter/Supprimer.
  11. Cliquez sur Oui lorsque vous êtes invité à confirmer la désinstallation.
