L'envers du décor

Cet exemple utilise un seul fichier : Counter.asp. Ce fichier utilise le contrôle DTC Recordset pour établir une connexion avec la table Counter dans la base de données Gallery. Une fois le jeu d'enregistrements ouvert, un nouvel enregistrement est ajouté à la table et des données sont écrites dans les champs appropriés :

function CounterRecordset_ondatasetcomplete(){
  fieldsArray = new Array("Remote_Host","Path_Info","HTTP_User_Agent");
  valuesArray = new Array(Request.ServerVariables("Remote_Addr"), Request.ServerVariables("Path_Info"), Request.ServerVariables("HTTP_User_Agent"));
  CounterRecordset.addImmediate(fieldsArray, valuesArray);
}

En utilisant le contrôle DTC Label, le nombre total d'enregistrements dans le jeu d'enregistrements permet ensuite d'afficher le nombre actuel d'accès à cette page.
