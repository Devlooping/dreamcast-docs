DirectDraw sample program -- Palettes
-----------------------------------------

* Purpose:
This sample demonstrates how to use palettized surfaces in WinCE for Dreamcast.
It loads a palettized (256 bpp) bitmap from the resource file, displays it on
the screen, and rotates through several different palettes.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
