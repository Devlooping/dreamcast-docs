DirectDraw sample program -- AlphaBlt
-----------------------------------------

* Purpose:
This sample shows how to do alphablting on WinCE for Dreamcast.  It displays
several alphablending surfaces fading in and out.

The Alphablt is accomplished by implementing the Alpha surface as a pair
of 3D triangles; this is hidden from the application behind helper functions
that handle scaling and drawing the image.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
