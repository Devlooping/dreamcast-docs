DirectDraw sample program -- BasicDDraw
-----------------------------------------

* Purpose:
This sample shows how to setup a basic DirectDraw application.  It creates
the front and back buffers, as well as an offscreen surface that it blts
to the screen.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
