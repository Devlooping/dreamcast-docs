MapleDevices sample program -- VibrationPack
-----------------------------------------

* Purpose:
This sample demonstrates how to use the Vibration pack on WinCE for 
Dreamcast.  When the sample is run, any controller with a vibration pack 
attached to it can activate the vibration pack in a variety of different
ways by pressing the controller's 'A', 'B', 'X', and 'Y' buttons.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
