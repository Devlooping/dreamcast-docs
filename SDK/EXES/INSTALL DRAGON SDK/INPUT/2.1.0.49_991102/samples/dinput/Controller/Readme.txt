DirectInput sample program -- Controller
-----------------------------------------

* Purpose:
This sample shows how to handle controllers (joysticks, steering wheels,
fishing controllers, etc) on WinCE for Dreamcast.  It handles multiple 
controllers, as well as plugging new controllers in dynamically.  It 
displays the current button and analog axis status on screen.

For sample code showing how to deal with the Keyboard, please see the 
keyboard sample in Samples\DInput\Keyboard.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
