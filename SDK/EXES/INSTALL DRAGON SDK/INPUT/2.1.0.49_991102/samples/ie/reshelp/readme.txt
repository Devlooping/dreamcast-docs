IE sample DLL -- ResHelp resource DLL
-----------------------------------------

* Purpose:
This sample shows how to create a resource DLL that contains html pages, gif files, etc. for
use with res protocol. For example, you can type res://reshelp.dll/help1.htm to load the page
called "help1.htm" from this DLL. This is an easy way to bundle all of your help files (or 
other html files) into one package.

* Required Files\Syntax:
None

* Used by
htmlsamp.exe
