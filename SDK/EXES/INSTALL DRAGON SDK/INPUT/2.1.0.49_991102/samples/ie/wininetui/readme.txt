IE sample DLL -- WININETUI Replaceable UI DLL
-----------------------------------------

* Purpose:
This sample shows how to create a replaceable UI DLL that allows you to customize certain
UI components in the HTML control. For example, you can change the basic authentication
dialog.

* Required Files\Syntax:
IE components

* Used by:
wininet.dll

* Registry entry:
[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings]
	"ReplaceableUIModule"="wininetui.dll"
