Demos:
===============
-D3DCheapBump:
Fast way of calculating Bumpmapping.

-D3DDragonFly:
Motion-blur effect using render to a texture.

-D3DFortune
Gigapixel fill-rate demo.

-D3DKnot.
Simple bump-map demo.

-D3DLightBeam
This demo shows how to create lights using cheap-shadows.

-D3DMarble.
Cheap shadows demo.

-D3DMaterials
Example that shows perception differences of different materials using bumpmapping.

-D3DScanner
Generic modifier volume demo.

-D3DSiberian
Punchtrough demo. 

-D3DTrilinear
Translucent and opaque trilinear.

-D3DWheel
'Realistic' bumpmapping demo.

-D3DVase
Multitexture.
This demo shows how to cut out an environment map using a 4444 texture.

-D3DMipmapping
 2x2, 4x4 and Fourier mipmapping

-D3DTextureFormats
 565, 1555, 4444, VQ and YUV texture formats.

*Notes: 
 Bumpmapping demos do not use the standard WinCE multitexture method for bumpmapping because
 different vertex colors are needed for different layers. This fact causes a penalty in performance 
 still not solved for v2.0.

 Use X to finish any demo but D3DMaterials that uses (START) to quit. 

 Build the demos using Retail as your Active Configuration.
