Direct3D sample program -- MipMapping
-----------------------------------------

* Purpose:
This sample demonstrates how to do mipmapping on WinCE for Dreamcast.  
It moves a mipmapping mesh towards and away from the viewer to demonstrate
the mipmapping affect.  It also moves a non-mipmapped version simultaneously
to show the difference that mipmapping provides.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
