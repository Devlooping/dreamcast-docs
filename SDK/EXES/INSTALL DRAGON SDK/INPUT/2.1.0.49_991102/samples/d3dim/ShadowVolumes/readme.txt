Direct3D sample program -- ShadowVolumes
-----------------------------------------

* Purpose:
This sample demonstrates how to use shadow volumes on WinCE for Dreamcast.
It displays a rotating sphere with a shadow cast on it by a single polygon.

For a volume sample that demonstrates how to modify texture selection based
on volume inclusion\exclusion, see the 'ComplexModifierVolumes' sample in
the D3DIM samples directory.

* Required Files\Parameters:
None

* To exit this sample, press the 'Send ESC' button in DCTool or
press <Esc> on the keyboard
