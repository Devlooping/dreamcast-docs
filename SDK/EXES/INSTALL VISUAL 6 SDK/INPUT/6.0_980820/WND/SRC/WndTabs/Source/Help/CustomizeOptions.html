Customize Options (Customize Dialog Box)

Overview

The Options tab of the Customize Dialog gives you control over various aspects of the WndTabs UI.

Options

Toolbar

Show ScreenTips on toolbars

Displays on-screen descriptions of toolbar buttons when the pointer pauses over them.

Show shortcut keys in ScreenTips

When selected, tool-tips show each command's assigned keyboard shortcuts.

Reset my usage data (Button)

When pressed, resets all the menus, toolbars and keyboard shortcuts to their initial state:

 

State After Reset
WndTabs Toolbar Single options menu with following commands: Reorder Tabs, Renumber Tabs, Addons Menu, Options, Customize, Help, and About. 
Tab Context Menu Reset according to the last selection specified through "Customize Using Command Groups" in the Options|Menus dialog.
Keyboard Shortcuts No shortcuts assigned.

Menus

Menu Animations

Changes the way menus are displayed when you click on a command.  For example, Slide Menu Animation causes the menu to display with a sliding motion.

Menu Shadows

When selected, menus display a light shadow.

Display Arrow Next To Menus...

Controls whether or not arrows are displayed next to menu buttons on the WndTabs toolbar.
