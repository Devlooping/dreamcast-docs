Introducing Window Tabs

"Window Tabs" (WndTabs) is an add-in for Microsoft Visual C++. It adds a docking bar of tabs reflecting all of the opened windows in the workspace. You can switch between the windows by simply clicking on the tab. The tabs have extensive right-click context menus full of useful functions.

The core WndTabs add-in is provided for free, with full source code. An extension module, with added functionality, is available at minimal cost. The registration fees paid for the extension module go to support the WndTabs project.
