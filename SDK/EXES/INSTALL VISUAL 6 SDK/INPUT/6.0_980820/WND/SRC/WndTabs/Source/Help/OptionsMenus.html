Customize|Menus & Toolbars (Options Dialog Box)

Menu Options

Shell context menu shortcut

This shortcut combination allows you to access a file's shell context menu directly.

Cascading Addons Menu

The addons menu (in both the main WndTabs menu and the tab context menus) displays commands exported by WndTabs addons.  This option controls whether addon commands are displayed in the main addon menu or in their own sub-menu.

Suppress menus for addons that don't export commands

With this option disabled, addons with no exported commands will have an empty sub-menu inside the Addons menu.  When enabled, this option will cause those sub-menus to disappear.

Customize Using Command Groups

Clicking on this button opens a context menu customization dialog.  The context menus are the menus you see when you right-click on a tab.
This customization dialog (one of two that are available - see below), allows you to customize the context menus by controlling four logical command groups (see here).  For each group, you may select on of the following options:   

None Will remove the entire group from the context menu
Regular The commands in the group will appear directly in the context menu
Cascading The commands in the group will appear in their own submenu inside the tab context menu.

Full Customizations

Clicking on this button opens the advanced customizations dialog.  This dialog differs from the Group Customization dialog in that it has access to all commands (even those not appearing in the four groups supported by the Group Customization dialog).  Also, it is not restricted to working with groups.
The advanced customizations dialog is available only to WndTabsExt users.  To find out more, click here.

Toolbar Options

Create native toolbar

Opens the native toolbar setup dialog.  The native DevStudio toolbar has the advantage of being able to dock anywhere within DevStudio.  However, only a handful of commands are available for this toolbar.

Customize WndTabs toolbar

This button opens the advanced customization dialog. Through this dialog, you can modify the WndTabs toolbar (the area to the left of the tabs).
