Try Before You Buy

If you want to check out the functionality of the extension module before purchasing a registration key, you may do so by going to the Options|Registration dialog box, and and clicking on "Start Free Trial".  You will then be prompted with two options:

  1. Automatically download the trial code - This requires that you be connected to the Internet.  Please note that this option doesn't work on all systems (for instance, it won't work through some firewalls).
  2. If the automatic code download doesn't work, or if you'd rather not use the automatic download, you can receive a code by e-mail by filling out the code request form on the WndTabs web site.

How Do I Register?

Registration is a very simple procedure. Just point your browser to the registration site, complete the form, and your registration code will be mailed to you in a matter of minutes.
