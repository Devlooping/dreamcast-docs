Languages (Options Dialog Box)

From this dialog you can select the language for WndTabs' dialogs and menus.  You can select any of the language packs currently installed, or download a new language pack site from the web site.  

Note that although some of the language changes appear immediately, you will have to restart DevStudio for all the changes to take effect.

WndTabs translations are created by volunteers.  If WndTabs has not been translated to your native language, and you'd like to do this, please contact me.
