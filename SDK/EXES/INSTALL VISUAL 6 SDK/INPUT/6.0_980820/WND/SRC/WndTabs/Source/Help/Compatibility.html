Compatibility With Other Add-Ins And Applications

Compatibility and Integration With the Workspace Whiz! Add-In

For the most complete integration with Workspace Whiz! (previously "Workspace Utilities"), you should use version 2.0 or above. The latest version can be downloaded from the Workspace Whiz! site, www.workspacewhiz.com . Highly recommended!

Also See: "Use Workspace Whiz! for header flipping" Option.

Compatibility With the Visual Assist Add-In

Visual Assist relies on many of same tricks as WndTabs to achieve its integration with DevStudio. Unfortunately, this can cause a few conflicts. If you want to use both WndTabs and Visual Assist, make sure to use Visual Assist version 3.0.3 build 1102 or above. This version can access internal WndTabs commands via the Add-In Communication Library in order to achieve peaceful coexistence.

NOTE: If you are using Visual Assist 3.x and are experiencing problems, try upgrading to version 4.  Many people have found version 4 much more stable than version 3 (especially when combined with WndTabs).

The Common Controls Library (COMCTL32.DLL)

Though not as common as they used to be, be advised that WndTabs will might work properly on system with older versions of the common controls library.  You must have the common controls library (COMCTL32.DLL) version 5.80 (or above) installed on your system for WndTabs to work properly. If you're still using old library versions, I highly recommend that you upgrade.

Incompatible Add-Ins/Applications

"Useful Managers" Add-In

WndTabs is completely incompatible with V1.0 of the "Useful Managers" add-in. If you wish to use "Useful Managers", make sure to upgrade to version 2.0, which is partially compatible with WndTabs.

Other Incompatible Software

The add-ins and applications on the following list are incompatible with WndTabs:

  • The "Sourcerer" Add-In (this addin seems to have been discontinued by it's authors).
