Installation
------------
To use this language pack follow these steps:
1. Copy the .dll file to the WndTabs installation directory (usually 
   "C:\Program Files\WndTabs").
2. Restart Visual C++.
3. In the WndTabs options dialog, pick "Language Selection" in the tree on
   the left hand side.
4. The new language should appear in the combo box.

Please note that this language pack will work ONLY with the version of
WndTabs specified at the top of this file.

-----------------------------------------------------------------------------

Instalacja
----------
Aby zainstalowac ten pakiet jezyka wykonaj nastepujace kroki:
1. Skopiuj plik .dll do katalogu instalacyjnego WndTabs (zwykle
   "C:\Program Files\WndTabs").
2. Zamknij i uruchom ponownie Visual C++.
3. W oknie dialogowym opcji WndTabs wybierz "Selekcja Jezyka" 
   (ang. "Language Selection") z drzewa wyboru po lewej stronie okna.
4. Nowy jezyk powinien byc widoczny w polu rozwijanym kombi.

Prosze zauwazyc, ze ten pakiet jezyka bedzie pracowal WYLACZNIE z wersja
WndTabs wyspecyfikowana u góry tego pliku.
