Installation
------------
To use this language pack follow these steps:
1. Copy the .dll file to the WndTabs installation directory (usually 
   "C:\Program Files\WndTabs").
2. Restart Visual C++.
3. In the WndTabs options dialog, pick "Language Selection" in the tree on
   the left hand side.
4. The new language should appear in the combo box.

Please note that this language pack will work ONLY with the version of
WndTabs specified at the top of this file.

Installazione
-------------
Per utilizzare questo language pack seguire i passi seguenti:
1. Copiare il file .dll nella directory di installazione di WndTabs
   (solitamente "C:\Programmi\WndTabs").
2. Riavviare il Visual C++.
3. Nel dialogo delle opzioni di WndTabs, evidenziare "Language Selection" 
   (cioè "Selezione Lingua") nella vista ad albero sulla parte sinistra.
4. La nuova lingua dovrebbe apparire nel combo box.

Si fa notare che questo language pack funzionerà SOLO con la versione di
WndTabs specificata all'inizio di questo file.
