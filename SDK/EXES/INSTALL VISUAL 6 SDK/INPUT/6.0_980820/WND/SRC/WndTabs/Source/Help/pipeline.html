Privacy Information

Some features of WndTabs send information over the Internet. "Others appear to send information (though they don't). " This section describes these features.

Automatic Trial Code Download

The automatic trial code download feature connects to the key generator on the WndTabs web site.  Like the version checking feature, this is done through HTTP, and both the version of WndTabs and OS are passed.

In order to prevent malicious use of this feature, WndTabs sends various parameters about your machine.  Though I won't go into the specifics of what is sent, I will mention what is not sent:

  1. No user names, e-mails or other identifications are sent.
  2. Nothing of an inventory nature is sent (e.g., the directory listing of your hard disk).

Some examples of things that might be sent are:

  1. The amount of RAM on your machine.
  2. The size or your hard disk.
  3. The number of hard disks on your machine.
  4. etc...

Again, the information transferred is not linked back to you, nor can it be used to track you or say anything about you.  It is only used to identify the machine you are using.

WndTabs Extension Module Initialization

WndTabs performs one other network based activity.  This involves checking the local network for other running copies of WndTabs, in order to enforce the license agreement.  In the capacity, WndTabs installs a listening port and sends a broadcast to the LAN.  The broadcast is performed using socket LAN broadcast mechanism, which does not exit the local area network.  To learn more, search for "2.2.4 broadcasting" in your MSDN.
