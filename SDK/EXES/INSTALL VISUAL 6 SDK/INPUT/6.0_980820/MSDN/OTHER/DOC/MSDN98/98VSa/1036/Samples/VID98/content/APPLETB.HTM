L'envers du décor

Les URL et le texte des grands titres sont stockés dans les balises <PARAM> qui sont lues par l'applet.

Chaque grand titre dispose de deux éléments dans ce fichier :
  • nom de l'exemple (affiché dans l'applet) ;
  • URL permettant d'accéder à l'exemple contenant plus d'informations.

Les paramètres répertoriés ci-après sont utilisés pour l'applet :

Paramètre Commentaire
BackColor Couleur de l'arrière-plan au format RVB
TextColor Couleur du texte du nom de l'exemple au format RVB
HiliteTextColor Couleur du texte du nom de l'exemple lorsqu'il est surligné au format RVB
ScrollDelay Délai séparant chaque défilement de pixel, mesuré en ms
MessageDelay Temps d'attente séparant les défilements des noms des exemples, mesuré en secondes
NumItems Nombre de noms des exemples
TextN Texte de l'exemple N
URLN URL du texte complet de l'exemple N
