L'envers du décor

Cet exemple utilise un fichier, GuestBk.asp. Il crée un formulaire HTML qui permet aux utilisateurs de taper leur nom ainsi que tous les commentaires voulus. Les informations fournies sont stockées dans une base de données. Les utilisateurs peuvent également afficher les commentaires entrés par les visiteurs précédents.

Le contrôle DTC Recordset permet d'établir une connexion avec la base de données. Si le formulaire est défini en entrée, vous voyez apparaître un formulaire HTML permettant d'effectuer des entrées. S'il est défini en sortie, le contrôle DTC Grid permet d'afficher le contenu de la table correspondant au livre d'or.
