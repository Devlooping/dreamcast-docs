L'envers du décor

Cet exemple utilise trois fichiers :
  • TLFish.htm contient le contrôle Timeline et d'autres fonctions permettant de gérer les objets sur la page.
  • FishLeft.gif est un fichier GIF animé d'un poisson qui va vers la gauche.
  • FishRght.gif est un fichier GIF animé d'un poisson qui va vers la droite.

Le contrôle Timeline déclenche des événements qui appellent des fonctions JavaScript permettant de contrôler le poisson et de modifier la position Z des algues.

Le code suivant est appelé par le contrôle Timeline lorsque le poisson est sur le point de se déplacer vers la droite. Avant que le poisson commence à nager, la position Z des algues est modifiée, l'image GIF animée du poisson nageant vers la gauche (FishLeft.gif) est masquée, l'image GIF animée du poisson nageant vers la droite (FishRght.gif) devient visible. Ensuite, la fonction appelle la méthode d'exécution du contrôle Timeline pour lancer l'animation.

function Swim_goRight()
{
   plant2.style.zIndex=-2 //Fait passer plant2 au second plan
   plant1.style.zIndex=1 //Fait passer plant1 au premier plan
   plant3.style.zIndex=1 //Fait passer plant3 au premier plan
   fishl.style.visibility="hidden"
   fishr.style.visibility="visible"
   Right.play()
}

Le contrôle Timeline appelle alors une autre fonction JavaScript qui déplace le poisson de 5 pixels toutes les 100 ms :

var moveInc = 5
function Right_frameRight()
{
   fishr.style.left = fishr.style.pixelLeft + moveInc
   fishl.style.left = fishl.style.pixelLeft + moveInc
}

Une fois que le poisson atteint le coin droit de l'écran, le contrôle Timeline utilise des fonctions similaires pour ramener le poisson vers la gauche.
