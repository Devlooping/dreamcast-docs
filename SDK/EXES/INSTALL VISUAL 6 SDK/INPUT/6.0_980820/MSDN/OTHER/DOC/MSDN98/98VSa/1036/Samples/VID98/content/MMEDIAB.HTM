L'envers du décor

Cet exemple n'utilise qu'un seul fichier :

  • MMedia.htm contient des améliorations pour les effets multimédias et des liens vers ces effets.

Les deux premières améliorations multimédias de ce fichier proviennent des balises HTML. La première permet de définir le fond sonore exécuté lors du chargement de cette page dans le navigateur :

<BGSOUND SRC=Utopia.wav>

La seconde amélioration exécute un fichier AVI de façon continue dans la page en utilisant la version améliorée de la balise <IMG> qu'offre Microsoft(r) Internet :

<IMG DYNSRC="Earthsml.avi" LOOP=Infinite>

L'amélioration multimédia suivante dans ce fichier utilise le contrôle Microsoft ActiveMovie, qui est directement intégré dans la page de façon à rendre inutile l'exécution d'un programme séparé.

Les paramètres suivants chargent le fichier multimédia, Clouds.avi, dans le lecteur et définissent les propriétés de l'objet pour déterminer comment l'objet est affiché et exécuté :

<OBJECT ID="ActiveMovie1" WIDTH=267 HEIGHT=155 CLASSID="CLSID:05589FA1-C356-11CE-BF01-00AA0055595A">
   <PARAM NAME="_ExtentX" VALUE="7038">
   <PARAM NAME="_ExtentY" VALUE="4101">
   <PARAM NAME="ShowDisplay" VALUE="0">
   <PARAM NAME="AllowHideDisplay" VALUE="0">
   <PARAM NAME="AllowHideControls" VALUE="0">
   <PARAM NAME="AutoStart" VALUE="0">
   <PARAM NAME="AutoRewind" VALUE="1">
   <PARAM NAME="Appearance" VALUE="0">
   <PARAM NAME="FileName" VALUE="Clouds.avi">
   <PARAM NAME="AllowChangeDisplayMode" VALUE="0">
   <PARAM NAME="DisplayForeColor" VALUE="0">
</OBJECT>
