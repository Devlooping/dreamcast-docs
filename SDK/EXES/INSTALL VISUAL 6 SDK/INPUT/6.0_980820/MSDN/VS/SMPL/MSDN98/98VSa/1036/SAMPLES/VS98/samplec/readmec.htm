Notes de mise à jour

Island Hopper ne respecte pas la casse - Il ne fonctionnera pas correctement dans une base de données SQL Server dont l'ordre de tri est défini sur le respect de la casse. Si vous voulez l'exécutez dans une base de données SQL Server avec respect de la casse, les modifications doivent être effectuées dans le code d'accès aux données des composants COM afin que toutes les requêtes utilisent des noms de tables et de champs correctement écrits.

Vous ne pouvez pas installer les niveaux A et C de l'exemple sur le même ordinateur. Avant d'installer un niveau de l'exemple, vérifiez si l'autre niveau est complètement désinstallé.

Les violations d'accès faisant référence à OLEChannelWnd:mtx.exe sont dues à une erreur du système d'exploitation. Vous pouvez télécharger un correctif logiciel pour résoudre cette erreur à partir du site suivant : ftp://ftp.microsoft.com/bussys/winnt/winnt-public/fixes/usa/NT40/hotfixes-postSP3/roll-up/

Installation et configuration du niveau C de Island Hopper News

Cette rubrique décrit la configuration requise et les instructions d'installation pour le niveau C de l'exemple Island Hopper News. Si vous souhaitez juste jeter un rapide coup d'oeil au code source, exécutez source.exe pour décompresser les fichiers source.

L'exemple Island Hopper News est fourni sur le CD-ROM de la bibliothèque MSDN. Vous pouvez copier les fichiers d'installation de l'exemple Island Hopper News sur votre disque dur ou installer les exemples directement à partir du CD-ROM de la bibliothèque MSDN.

Pour copier les fichiers d'installation de l'exemple Island Hopper News sur votre disque dur

  1. Insérez le CD-ROM de la bibliothèque MSDN dans votre lecteur de CD-ROM et double-cliquez sur Installation.

  2. Cliquez sur Ajouter/Supprimer pour ajouter de nouveaux composants.

  3. Sélectionnez Exemple Island Hopper News, puis cliquez sur Continuer.

Les fichiers de l'exemple Island Hopper News sont copiés dans Program Files\Microsoft Visual Studio\MSDN98\98VS\1033\Samples\Vs98\SampleC si vous acceptez l'emplacement par défaut du dossier d'installation.

Configuration requise

Le niveau C de l'exemple Island Hopper News nécessite au moins un ordinateur compatible avec la configuration minimale requise pour Visual Studio. Pour plus d'informations sur les configurations logicielle et matérielle requises, voir la rubrique en ligne correspondante dans la bibliothèque MSDN Visual Studio 6.0. Vous pouvez installer le niveau C sur 4 ordinateurs au maximum, chaque ordinateur jouant un rôle différent. Les rôles peuvent être répartis de la manière suivante :

  • Ordinateur du développeur : ordinateur utilisé pour travailler sur le code source.

  • Ordinateur client : ordinateur où sont installés les clients Windows et Web de l'application exemple.

  • Ordinateur serveur : ordinateur où sont déployés les composants contenant les règles d'entreprise ainsi que les pages ASP qui constituent le client Web.

  • Ordinateur base de données : ordinateur où est installée la base de données de l'application exemple.

Vous avez la possibilité d'utiliser un ordinateur distinct pour chaque rôle ou de cumuler plusieurs rôles sur un même ordinateur ; les quatre rôles peuvent être regroupés sur un ordinateur très puissant. Chaque rôle a son propre programme d'installation. Exécutez les programmes d'installation sur les ordinateurs que vous avez choisis pour chaque rôle. Si vous avez décidé d'utiliser un seul ordinateur, exécutez les quatre programmes d'installation sur cet ordinateur.

Configuration logicielle requise

Avant d'installer le niveau C, vous devez avoir installé les logiciels suivants :

Ordinateur client Ordinateur du développeur Ordinateur serveur Ordinateur base de données
Internet Explorer 4.0 ou version ultérieure Visual Studio 6.0, Édition Entreprise

Windows NT Option Pack, version 4.0

Autres informations

Regroupez les informations suivantes avant de commencer la procédure d'installation, car elles vous seront demandées par les programmes d'installation.

  • Nom de votre serveur IIS/MTS

  • Emplacement serveur où installer le contenu web de l'exemple

  • Nom de l'ordinateur gérant la base de données

  • ID utilisateur et mot de passe SQL Server

Instructions d'installation

Important :   Si vous installez tous les exemples sur le même ordinateur, veillez à exécuter les procédures suivantes dans l'ordre où elles sont présentées et à lancer le programme d'installation du serveur en dernier.

Les programmes d'installation sont des fichiers .zip auto-extractibles. Il suffit de double-cliquer sur l'un d'eux pour décompresser les fichiers nécessaires et démarrer automatiquement le programme d'installation.

Les procédures suivantes supposent que vous installez l'exemple directement à partir du CD-ROM de la bibliothèque MSDN. Si vous avez déjà copié les fichiers de l'exemple sur votre disque dur, remplacez l'emplacement mentionné dans les étapes par celui que vous avez effectivement utilisé.

Pour installer les clients Island Hopper News

  1. Insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM de l'ordinateur client.

  2. Naviguez jusqu'à Samples\Vs98\SampleC.

  3. Double-cliquez sur Client.exe pour décompresser et exécuter le programme d'installation client.

Pour installer le code source Island Hopper News

  1. Insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM de l'ordinateur du développeur.

  2. Naviguez jusqu'à Samples\Vs98\SampleC.

  3. Double-cliquez sur Source.exe pour décompresser et exécuter le programme d'installation du code source.

Pour installer les composants serveur de Island Hopper News

  1. Insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM de l'ordinateur serveur.

  2. Naviguez jusqu'à Samples\Vs98\SampleC.

  3. Double-cliquez sur Server.exe pour décompresser et exécuter le programme d'installation du serveur.

Pour installer la base de données Island Hopper News

  1. Insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM de l'ordinateur de la base de données.

  2. Naviguez jusqu'à Samples\Vs98\SampleC.

  3. Double-cliquez sur Database.exe pour décompresser et exécuter le programme d'installation de la base de données.

Le niveau C de Island Hopper News est maintenant installé sur votre système. Le code source se trouve sur l'ordinateur du développeur, dans Program Files\IslandHopper\ScenarioC. Vous pouvez exécuter le client Windows ou le client Web de l'application.

Pour démarrer le client Windows :

  1. Cliquez sur Démarrer sur l'ordinateur client.
  2. Pointez sur Programmes.
  3. Pointez sur Island Hopper.
  4. Cliquez sur Island Hopper C.

Pour démarrer le client Web :
  1. Démarrez Internet Explorer sur l'ordinateur client.
  2. Dans la zone Adresse, tapez le nom du serveur Web où vous avez installé l'exemple Island Hopper et le dossier du client Web (islandhopperc). Par exemple, http://monserveur/islandhopperc.

Pour plus d'informations, consultez les rubriques en ligne suivantes dans la bibliothèque MSDN Visual Studio 6.0 :

  • Exemple Island Hopper News
  • Rechercher et exécuter le niveau C
  • Spécifications du niveau C

Instructions de désinstallation

Aux programmes d'installation du client, du code source et du serveur sont associés les programmes de désinstallation correspondants. Il n'existe pas de programme de désinstallation pour la base de données. Si vous utilisez un même ordinateur pour plusieurs rôles, vous devez procéder à la désinstallation dans l'ordre inverse de celui de l'installation. Par exemple, vous pouvez exécuter les programmes de désinstallation dans l'ordre suivant :

  1. Désinstallation du serveur

  2. Désinstallation du code source

  3. Désinstallation du client

Si vous utilisez plusieurs ordinateurs pour cet exemple, vous devez exécuter les programmes de désinstallation sur chaque ordinateur. Par ailleurs, si vous avez installé les clients ou le code source sur un ordinateur différent du serveur, il faut désinstaller des composants supplémentaires sur les ordinateurs client et développeur.

Les procédures suivantes expliquent comment désinstaller l'exemple Island Hopper News sur un ou plusieurs ordinateurs.

Pour désinstaller Island Hopper News sur un ordinateur unique

Procédez de la manière suivante pour supprimer l'exemple Island Hopper News si vous avez installé tous les éléments sur un même ordinateur.

  1. Insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM.

  2. Naviguez jusqu'à Samples\Vs98\SampleC.

  3. Double-cliquez sur RempackC.exe pour supprimer les packages Island Hopper de Microsoft Transaction Server.

  4. Cliquez sur Démarrer, pointez sur Paramètres, puis cliquez sur Panneau de configuration.

  5. Double-cliquez sur Ajout/Suppression de programmes.

  6. Cliquez sur Installation/Désinstallation, si nécessaire.

  7. Sélectionnez Island Hopper Scenario C Server, puis cliquez sur Ajouter/Supprimer.

  8. Sélectionnez Island Hopper Scenario C Source, puis cliquez sur Ajouter/Supprimer.

  9. Sélectionnez Island Hopper Scenario C Client, puis cliquez sur Ajouter/Supprimer.

Pour désinstaller Island Hopper News sur plusieurs ordinateurs

Procédez de la manière suivante si vous avez installé le client ou le code source sur un ordinateur autre que le serveur.

  1. Sur l'ordinateur client, cliquez sur Démarrer, pointez sur Paramètres, puis cliquez sur Panneau de configuration.

  2. Double-cliquez sur Ajout/Suppression de programmes.

  3. Cliquez sur Installation/Désinstallation, si nécessaire.

  4. Sélectionnez Remote Application - IsleHop Accounting (remove only), puis cliquez sur Ajouter/Supprimer.

  5. Sélectionnez Remote Application - IsleHop Classifieds (remove only), puis cliquez sur Ajouter/Supprimer.

  6. Sélectionnez Remote Application - IsleHop Utilities (remove only), puis cliquez sur Ajouter/Supprimer.

  7. Sélectionnez Island Hopper Scenario C Client, puis cliquez sur Ajouter/Supprimer.

  8. Répétez ces étapes sur l'ordinateur où vous avez installé le code source.

    Remarque :   Si vous avez installé le client et le code source sur le même ordinateur, supprimez d'abord Island Hopper Scenario C Source à l'étape 7, puis Island Hopper Scenario C Client.

  9. Sur l'ordinateur serveur, insérez le CD-ROM 1 de la bibliothèque MSDN dans le lecteur de CD-ROM.

  10. Naviguez jusqu'à Samples\Vs98\SampleC.

  11. Double-cliquez sur RempackC.exe pour supprimer les packages Island Hopper de Microsoft Transaction Server.

  12. Cliquez sur Démarrer, pointez sur Paramètres, puis cliquez sur Panneau de configuration.

  13. Double-cliquez sur Ajout/Suppression de programmes.

  14. Cliquez sur Installation/Désinstallation, si nécessaire.

  15. Sélectionnez Island Hopper Scenario C Server, puis cliquez sur Ajouter/Supprimer.
