Exemple Island Hopper News

   

L'application exemple de Visual Studio 6.0, Island Hopper News, illustre la façon dont vous pouvez concevoir et structurer des applications à l'aide de la plate-forme Microsoft et de la suite Visual Studio.

Island Hopper News est un journal en ligne fictif, conçu pour être publié sur le World Wide Web. Il présente des actualités locales, nationales et internationales, mais également des résumés de rencontres sportives locales et une rubrique d'annonces classées. Seule une partie de l'application Island Hopper News est mise en oeuvre en tant qu'application exemple de Visual Studio 6.0 : la section des annonces classées. Vous n'y trouverez pas toutes les fonctionnalités que vous souhaiteriez. L'exemple a été créé ainsi pour vous offrir la meilleure base de travail possible : des éléments à partir desquels vous pourrez développer une application répondant à vos besoins et à vos objectifs.

Cet application exemple est disponible en deux versions appelées « niveaux ». Chaque niveau correspond à un degré d'expérience et de connaissances différent de l'utilisateur. Le niveau A est une version à échelle réduite de l'application, parfaite pour un intranet d'entreprise. Le niveau C est une application commerciale qui convient au déploiement sur Internet. Les deux niveaux comprennent des applications client, des composants d'entreprise et d'accès aux données et des bases de données.

Vous pouvez installer et travailler avec n'importe quel niveau de l'exemple. Le niveau C suppose que vous possédez les connaissances et compétences acquises au cours du niveau A.

L'application exemple se trouve sur le CD-ROM de la bibliothèque MSDN, dans Disk1\Samples\Vs98. Pour copier les fichiers de l'exemple sur votre disque dur, exécutez le programme d'installation de la bibliothèque MSDN, choisissez l'option d'installation personnalisée et activez la case à cocher Island Hopper Samples.

Island Hopper News : Niveau A

Séléctionnez ce niveau si vous souhaitez examiner la forme la plus simple de l'exemple Island Hopper News. Le niveau A est une application élémentaire qui vous permet de consulter et de publier des annonces classées.

Technologies

Le niveau A est une application client/serveur comprenant deux extrémités frontales ou clients : un client Windows standard développé à l'aide de Visual Basic et un client Web développé à l'aide de Visual InterDev. Dans le niveau A, les données constituant les annonces classées sont stockées dans une base de données Jet (moteur de base de données utilisé par Microsoft Access) dont l'extension est .mdb. Ce niveau propose également une implémentation simple de composants COM (modèle d'objet composant) pour illustrer l'encapsulation des règles d'entreprise. Ces composants ont été développés en Visual Basic.

Installation

Les instructions d'installation et de configuration pour le niveau A de Island Hopper News sont fournies par le fichier lisez-moi du niveau A. Ce fichier se trouve à l'un des deux endroits suivants :

  • Si vous n'avez pas copié les fichiers de l'exemple sur votre lecteur local à partir du CD-ROM de la bibliothèque MSDN, suivez le chemin d'accès Disk1\Samples\VS98\samplea\readmea.htm sur ce CD-ROM.

  • Si vous avez copié les fichiers de l'exemple sur votre disque local, suivez le chemin Program Files\Microsoft Visual Studio\MSDN98\98VS\1036\Samples\VS98\SampleA\readmea.htm.

Island Hopper News : Niveau C

Sélectionnez ce niveau si vous souhaitez voir la transition entre une application Web interne et une application Internet publique. Le niveau C montre comment Island Hopper Publishing (la société mère de Island Hopper News) a transféré son application d'annonces classées sur son site Web externe. Il intègre les notions de suivi de clientèle et de facturation dans le domaine des annonces classées (et d'autres secteurs d'activité de la société). Pour les besoins de l'exemple, une partie seulement d'une application complète est illustrée : il s'agit de la section contenant les annonces classées.

Technologies

Le niveau C est une application d'entreprise complète qui tire parti de toutes les fonctionnalités de Microsoft Transaction Server (MTS), Internet Information Server (IIS) et SQL Server. Comme le niveau A, il s'agit d'une application client/serveur à deux clients : un client Windows standard développé à l'aide de Visual Basic et un client Web développé avec Visual InterDev. Dans le niveau C, les deux clients desservent toutefois des utilisateurs différents :

  • Les employés de la société Island Hopper Publishing utilisent le client Windows pour la gestion du fichier des clients, le suivi des factures et des paiements.

  • Les clients de Island Hopper Publishing utilisent le client Web pour explorer et passer des annonces classées dans le journal Island Hopper News.

Le niveau C illustre une implémentation plus complexe des composants COM pour l'encapsulation des règles d'entreprise et de l'accès aux données. Ces composants ont été développés à l'aide de Visual Basic pour la plupart d'entre eux, avec Visual C++ dans quelques cas.

Installation

Les instructions d'installation et de configuration du niveau C de Island Hopper News sont fournies par le fichier lisez-moi du niveau C. Ce fichier se trouve à l'un des deux endroits suivants :

  • Si vous n'avez pas copié les fichiers de l'exemple sur votre lecteur local à partir du CD-ROM de la bibliothèque MSDN, suivez le chemin Disk1\Samples\VS98\samplec\readmec.htm sur ce CD-ROM.

  • Si vous avez copié les fichiers de l'exemple sur votre disque dur, suivez le chemin Program Files\Microsoft Visual Studio\MSDN98\98VS\1033\Samples\VS98\SampleC\readmec.htm.
