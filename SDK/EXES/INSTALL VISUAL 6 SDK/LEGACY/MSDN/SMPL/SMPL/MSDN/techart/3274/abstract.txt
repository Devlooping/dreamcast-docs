 
 
 YAHU: Analyzes Executable Files
 
 The YAHU sample application implements a generic and
 extensible .EXE header utility with a few value-added
 functionalities, such as run-time monitoring of
 Microsoft(R) Win32(R) applications.
 
 For more information on YAHU, refer to the "YAHU, or Yet
 Another Header Utility" technical article in the Microsoft
 Development Library.
 
 YAHU was built with Microsoft Visual C++(TM) version 2.0,
 and tested under Microsoft Windows NT(TM) version 3.5.
 
 KEYWORDS: CD11; debug API; DLL loading; executable file
 format; process list
