
HOUSE & APPLIANCES: Demonstrate Aggregation of Objects
The 32-bit HOUSE4 sample application and its associated
APPLIANCES.DLL accompany the "MFC/COM Objects 4:
Aggregation"msdn_house4 technical article in the
Microsoft(R) Development Library. This sample shows how COM
objects use aggregation of other COM objects.
Note that to run the HOUSE4 sample, you must first register
the COM objects in the registry. Copy HOUSE.EXE and
APPLIANCES.DLL to your local disk, then edit APPLIANCES.REG
to show the path to where you installed APPLIANCES.DLL. Type
"REGEDIT -s APPLIANCES.REG" (without the quotation marks) in
a command window to create the registry entries.
Both the COM object and the application were built using
Microsoft Visual C++(TM) version 2.1, and tested under
Microsoft Windows NT(TM).
