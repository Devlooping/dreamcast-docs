
Interp: Shows Stretching of 256-Color Images with Pixel
Interpolation
Interp accompanies the "Stretching 256-Color Images Using
Interpolation"msdn_interp technical article in the
Microsoft(R) Development Library.
Interp demonstrates how a 256-color palletized image can be
stretched by a factor of 2 with interpolation. The
interpolation is done via a lookup table and provides
excellent performance.
This sample application was built with Microsoft Visual C++
version 2.1, and tested under Windows NT (TM).
