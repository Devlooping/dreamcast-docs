SPLDEMO: Uses Microsoft Schedule+ 95 as an OLE Automation 
Server 
The SPLDEMO sample application demonstrates the use of Microsoft(R) Schedule+ 95 as an OLE 
Automation server. This sample uses Microsoft Visual Basic(R) version 4.0 as the OLE Automation 
controller. Developers can use this sample to gain a better understanding of how to implement OLE 
Automation with Microsoft Schedule+ 95.
SPLDEMO is a companion sample for the "Using Microsoft OLE Automation Servers to Develop 
Solutions" technical article in the Microsoft Development Library.
Note that you must have Microsoft Schedule+ 95 and its associated type library fully installed on your 
system to run this sample.
The SPLDDEMO sample was built with the Enterprise Edition of Microsoft Visual Basic version 4.0 and 
tested on Microsoft Windows(R) 95.
KEYWORDS: CD14; OLE; VBA
