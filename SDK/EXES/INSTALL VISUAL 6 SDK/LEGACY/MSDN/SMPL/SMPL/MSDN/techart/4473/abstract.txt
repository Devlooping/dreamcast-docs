DIALIT: Uses the CTapiConnection Class
DIALIT implements a simple Telephony Application Programming Interface (TAPI) functionality in a 
Microsoft(R) Foundation Class Library-based (MFC) class for the Microsoft Windows(R) 95 operating 
system. It provides an easy way to initialize TAPI, connect, and make a voice-based telephone call through 
TAPI.
DIALIT is a companion sample for the "Creating a TAPI Connection Using CTapiConnection" technical 
article in the Microsoft Development Library.
The DIALIT sample was built with Microsoft Visual C++(TM) version 2.2 and MFC version 3.1, and was 
tested on Windows 95.
