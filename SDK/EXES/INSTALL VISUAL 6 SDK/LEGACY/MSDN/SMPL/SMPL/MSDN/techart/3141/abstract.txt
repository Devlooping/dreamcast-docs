 
 
 GENERIC: Illustrates Minimal Win32 Functionality
 
 GENERIC illustrates the minimum functionality that should
 be expected of a well-behaved Microsoft(R) Win32(R)
 application. Otherwise, this sample application has
 virtually no substantial functionality of its own, making
 it easy for you to use this as a starting point for your
 own application development or investigation of the Win32
 application programming interface (API).
 
 GENERIC was built with Microsoft Visual C++(TM) version
 2.0 and the Microsoft Windows(R) 95 Software Development
 Kit (SDK). It was tested under Microsoft Windows NT(TM)
 version 3.5.
