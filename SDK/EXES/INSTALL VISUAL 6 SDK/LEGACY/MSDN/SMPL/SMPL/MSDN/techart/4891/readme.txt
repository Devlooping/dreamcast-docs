Introduction
------------

This folder contins four files that can help you in testing and debugging add-ins using
Microsoft Access for Windows 95 version 7.0:

 - Addinhlp.doc - A Microsoft Word document describing six 
   scenarios of common problems that you may encounter while 
   developing add-ins. 
 
 - USysReg.doc - A Microsoft Word document containing a Help 
   topic from Microsoft Access 7.0. The topic discusses how
   to create a USysRegInfo table to install custom add-ins.

 - Wzaddin.mda - A Microsoft Access 7.0 library database with
   several sample add-ins.

 - Wzcodlib.mda - A Microsoft Access 7.0 library database that
   contains a function called by a sample add-in from Wzaddin.mda.

For more information about these files, please see the Addinhlp.doc file.

Installation
------------

To install the contents of this folder, copy the enclosed files to the following folders:

Windows 95
==========

Filename       Folder
-------------------------------------------- 
Addinhlp.doc   c:\my documents  
USysReg.doc    c:\my documents
Wzaddin.mda    c:\msoffice\access
Wzcodlib.mda   c:\program files\common files

Windows NT
==========

Filename       Folder
-------------------------------------------- 
Addinhlp.doc   c:\msoffice\access  
USysReg.doc    c:\msoffice\access
Wzaddin.mda    c:\msoffice\access
Wzcodlib.mda   c:\winnt35\msapps

Once you copy these files, you can print the Addinhlp.doc and USysReg.doc files from a word
processor, such as Microsoft Word. The Addinhlp.doc file uses the sample library databases,
Wzaddin.mda and Wzcodlib.mda, to demonstrate common problems in testing and debugging add-ins.
