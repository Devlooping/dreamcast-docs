 
 
 EasyTex: Demonstrates OpenGL Texture Mapping
 
 The EasyTex sample application demonstrates how to texture-
 map OpenGL(TM) objects with Microsoft(R) Windows(R) device-
 independent bitmaps (DIBs). EasyTex uses
 GLlib.DLLmsdn_sample_gllib sample application (available
 in the Microsoft Development Library), which contains a
 class library for OpenGL.
 
 The "OpenGL VII: Scratching the Surface of Texture
 Mapping"msdn_gl7 technical article in, also available in
 the Microsoft Development Library, discusses the texture-
 mapping techniques illustrated by EasyTex.
 
 EasyTex was built with Microsoft Visual C++(TM) version
 2.1 and the Microsoft Foundation Class Library (MFC)
 version 3.1, and tested under Microsoft Windows NT(TM)
 version 3.5 on the Intel(R), MIPS(R), and DEC(R) Alpha
 AXP(TM) platforms.
