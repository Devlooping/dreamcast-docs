 
 
 COMPPR: Demonstrates Fundamental OLE Functionality
 
 COMPPR is a companion for "The Component Object Model:
 Technical Overview" article in the Microsoft(R)
 Development Library, and demonstrates many of the concepts
 this article introduces.
 
 COMPPR was built with Microsoft Visual C++(TM) version 2.0
 and the Microsoft Win32(R) Software Development Kit (SDK),
 and was tested under Microsoft Windows NT(TM) version 3.5.
 
 KEYWORDS: CD10
