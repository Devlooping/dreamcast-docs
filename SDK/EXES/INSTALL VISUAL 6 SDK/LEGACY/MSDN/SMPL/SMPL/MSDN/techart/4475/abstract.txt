SICLIENT: Calls COM Objects with Smart Interface Pointers
SICLIENT is a companion sample application for the "Calling COM Objects with Smart Interface Pointers" 
technical article in the Microsoft(R) Development Library.
The SICLIENT sample demonstrates how to use smart interface pointers. Smart interface pointers are smart 
pointers designed to be used with Component Object Model (COM) interfaces. Smart interfaces are 
implemented in SmartInterface.h. SICLIENT calls the ISimple interface implemented in SimpleObject.dll.
Before SICLIENT will work, you must register the SimpleObject object. First, edit the paths in 
SimpleObject.reg to point to the location of SimpleObject.dll. Then double-click SimpleObject.reg to add it 
to the registry.
This project builds both SIClient.exe and SimpleObject.dll, and requires Microsoft Visual C++(TM) 
version 4.0 to run.
The SICLIENT sample was built with Microsoft Visual C++ version 4.0, and tested on Microsoft Windows 
NT(TM) version 3.51.
