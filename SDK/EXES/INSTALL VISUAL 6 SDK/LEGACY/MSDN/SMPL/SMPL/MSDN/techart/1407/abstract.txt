
WALKER: Views Process Virtual Memory
WALKER lets you view information about the virtual memory of
a process.
WALKER is a Microsoft(R) Win32(R) Software Development Kit
(SDK) sample. It was built with this SDK and Microsoft
Visual C++(TM) version 2.1, and tested under Microsoft
Windows NT(TM) version 3.51.
