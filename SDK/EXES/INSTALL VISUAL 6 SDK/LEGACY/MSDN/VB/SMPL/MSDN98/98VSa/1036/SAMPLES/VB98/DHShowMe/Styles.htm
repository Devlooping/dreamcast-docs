Changement dynamique de styles

Voici une démonstration de quelques unes des propriétés typographiques et de contrôle associées aux éléments de texte d'une page HTML, comme codé dans Visual Basic. Chaque élément de texte peut être transformé de diverses façons par modification de sa police. Par exemple, vous pouvez :

  • Souligner
  • Changer de couleur
  • Agrandir et réduire
  • Barrer

ou modifier d'autres façons l'apparence de vos éléments typographiques.

Déplacez la souris sur les puces ci-dessus pour voir ces effets en action.
