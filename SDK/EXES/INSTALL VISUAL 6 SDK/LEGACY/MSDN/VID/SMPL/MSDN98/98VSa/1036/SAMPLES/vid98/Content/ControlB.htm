L'envers du décor

Cet exemple utilise deux fichiers : DTCTest.asp et DTCTest.htm. Ces deux fichiers accomplissent la même tâche, mais la version ASP exige l'actualisation de la page chaque fois qu'une valeur est modifiée. Comme la page HTML utilise DHTML (Dynamic HTML), l'actualisation de la page n'est pas nécessaire lorsqu'une valeur est modifiée.
