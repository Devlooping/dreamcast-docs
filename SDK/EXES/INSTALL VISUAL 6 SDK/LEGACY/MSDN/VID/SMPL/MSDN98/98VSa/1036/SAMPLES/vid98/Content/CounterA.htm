Cet exemple relève chaque connexion à une page ASP et consigne les informations relatives à l'utilisateur qui consulte la page. L'adresse IP, le chemin de la page visualisée et le navigateur client sont tous enregistrés dans la table Counter de la base de données Gallery. Ces valeurs permettent ensuite d'exécuter des rapports afin de connaître l'activité d'une page.
