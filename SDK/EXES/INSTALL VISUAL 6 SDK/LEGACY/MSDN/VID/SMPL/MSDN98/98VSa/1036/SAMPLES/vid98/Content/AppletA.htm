Une applet Java nommée CoolHeadLines permet de faire défiler les noms des exemples dans cette page Web exemple. Lorsqu'un utilisateur clique sur le nom d'un exemple, il est dirigé vers l'URL appropriée et obtient des informations supplémentaires sur l'exemple concerné.

Cet exemple a été emprunté à l'exemple en ligne « ExAir » de Microsoft(r) Internet Information Server 4.0.
