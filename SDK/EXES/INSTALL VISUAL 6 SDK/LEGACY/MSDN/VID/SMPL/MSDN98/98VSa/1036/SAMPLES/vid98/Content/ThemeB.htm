L'envers du décor

Cet exemple utilise trois fichiers HTML :
  • NoTheme.htm est une simple page HTML qui utilise trois niveaux différents de titres et de puces.
  • Theme1.htm utilise les mêmes balises HTML que NoTheme.htm mais le thème Espace lui est appliqué.
  • Theme2.htm utilise le thème Nature.

Lorsqu'un thème est appliqué à une page HTML ou au projet Web entier, les fichiers sont copiés dans le dossier Themes du projet. La plupart des fichiers sont au format graphique, mais il existe aussi quelques feuilles de style en cascade (CSS).

Le fichier .css est la clé du thème. Les thèmes de cet exemple utilisent différentes feuilles de style pour créer le thème :

<link REL="stylesheet" TYPE="text/css" HREF="_Themes/travel/COLOR0.CSS" VI6.0THEME="barcode">
<link REL="stylesheet" TYPE="text/css" HREF="_Themes/travel/GRAPH0.CSS" VI6.0THEME="barcode">
<link REL="stylesheet" TYPE="text/css" HREF="_Themes/travel/THEME.CSS" VI6.0THEME="barcode">
<link REL="stylesheet" TYPE="text/css" HREF="_Themes/travel/CUSTOM.CSS" VI6.0THEME="barcode">

Le premier fichier lié (COLOR0.CSS) définit les couleurs des liens (ordinaires, actifs et visités), les six niveaux différents de titres (H1-H6), les couleurs d'activation et d'ombre de la table et la couleur de fond du corps.

Le deuxième fichier lié (GRAPH0.CSS) définit les informations d'image et de police pour la bannière et les contrôles de navigation non graphiques.

Le troisième fichier lié (THEME.CSS) définit la famille de polices du corps, les six niveaux de titre et l'image d'arrière-plan.

Tous les contrôles de navigation graphique se trouvent dans le fichier lié final (CUSTOM.CSS) avec tous les niveaux de puces.
