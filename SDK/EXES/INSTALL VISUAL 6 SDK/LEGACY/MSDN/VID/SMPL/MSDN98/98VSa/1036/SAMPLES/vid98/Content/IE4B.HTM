L'envers du décor

Cet exemple utilise cinq fichiers :

  • IE4.asp est le principal fichier qui définit le jeu de cadres.
  • IE4.htm est un exemple simple de DHTML qui apparaît dans le cadre d'en-tête.
  • Main.htm remplit le cadre principal.
  • Filter.htm illustre l'utilisation des filtres.
  • Trans.htm présente différents effets de transition.

IE4.asp définit le jeu de cadres de cet exemple pour une navigation aisée.

IE4.htm utilise DHTML pour créer un effet visuel lorsque la souris passe sur un lien. Pour ce faire, il suffit de définir quelques styles différents pour la balise <A> :

<style>
A.Item {
   cursor: hand;
   font-family: verdana;
   font-size: 12;
   font-style: normal;
   background-color: black;
   color: white;
   text-decoration: none
}
A.Highlight {
   cursor: hand;
   font-family: verdana;
   font-size: 12;
   font-style: italic;
   background-color: white;
   color: black;
   text-decoration: none
}
</style>

Pour passer d'un style à l'autre, utilisez JavaScript pour modifier le style appliqué selon que la souris s'approche ou s'éloigne des liens :

<script LANGUAGE=JavaScript>
function rollon() {
if (window.event.srcElement.className == "Item") {
window.event.srcElement.className = "Highlight";
}
}

Rollover.onmouseover = rollon;

function rolloff() {
if (window.event.srcElement.className == "Highlight") {
window.event.srcElement.className = "Item";
}
}

Rollover.onmouseout = rolloff;
</script>

<input CLASS=clsbtn value="FlipH" type=button name=filt onclick="theImg.style.filter = 'fliph(enabled=1)'; progress.innerText = 'Flip Horizontal (fliph): Flips the visible pixels in the horizontal axis.';">

Trans.htm permet de passer en revue les différents effets de transition que vous pouvez appliquer aux objets. Dans cet exemple, une image apparaît sur du texte et effectue un fondu à l'ouverture et à la fermeture avec différents effets de transition : AB.filters.item(0).Transition = curtrans

AB se rapporte à la balise <DIV> qui définit les objets qui vont être concernés :

<DIV ID="AB" STYLE="POSITION:ABSOLUTE; WIDTH:300; HEIGHT:360; Left:45%; Top:15%;background-color:black; FILTER:revealTrans(Duration=3.0, Transition=1)">

curtrans est une variable de code JavaScript qui passe en revue les différents effets de transition :

curtrans = curtrans + 1
if curtrans > numtrans then curtrans = 0 end if
