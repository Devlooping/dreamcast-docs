Introduction

Que vous vouliez accéder à une base de données, écrire des scripts sur le client ou le serveur ou utiliser de nouvelles fonctionnalités de Internet Explorer 4.0, cette galerie d'exemples est pour vous. Cette galerie est divisée en quatre grandes sections : Base de données, Scripts, Exemples intégrés et Exemples Web.

Les exemples de bases de données montrent comment extraire et enregistrer des informations dans une base de données à l'aide des contrôles DTC fournis avec Visual InterDev. Les exemples de scripts indiquent comment utiliser VBScript et JavaScript sur le client et le serveur pour lire, écrire et vérifier l'entrée client. Les exemples intégrés décrivent comment incorporer des éléments qui pourraient être utilisés dans plusieurs applications. Les exemples Web expliquent comment améliorer votre application Web avec des thèmes, des éléments multimédias et des publicités.
